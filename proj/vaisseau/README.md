# Mini projet avec pyxel

A partir de la 1ère version jouable réalisée par vous, ou [proj06_JOUABLE_1.py](proj06_JOUABLE_1.py)
- ajouter des fonctionnalités
  - score
  - gestion des collisions
  - points de vie
  - possibilité de rejouer après la fin de la partie
  - niveaux de difficulté...
  - autres idées
- vous pourrez utiliser un dictionnaire pour stocker les nouvelles valeurs utiles pour le jeu
  - pensez à sauvegarder régulièrement votre travail (en numérottant chaque version mise au point)
  - pensez à TESTER très régulièrement votre code
- le code final est à remettre dans le dossier à votre nom
